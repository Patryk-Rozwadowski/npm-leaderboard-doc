Leaderboard npm package  

v 1.0.0

Description

A simple, lightweight and generic leaderboard module created in vanilla javascript. You can create multiple leaderboards on one page and set different options for each. Visit the Github Page to see the module API.
NPM
npm install leaderboard
YARN
yarn add leaderboard
CDN
https://unpkg.com/leaderboard@1.0.0/
